HackInOut Coding Event


This challenge lets you improve your coding skills with games. It all starts in the IDE dedicated for this game,
where you will code and test new ideas.This is your mission statement.
Solo and multiplayer coding games are turn-based: at each turn, your program gets new inputs and must output the action.
Choose a programming language: You can choose a language you already know or in which you are comfortable with. It doesnt matter even if you dont know any language completely, but as long as you know the basic syntax for forand while loops, and if statements,
you should be just fine
Time to code:Use your logical skills along with your programming skills , to solve all the three games, and if you do it first, you loot the prize.
Lets check your code: Games have test cases to make sure you have solved it properly. Run your code by playing a test case. If the test case runs without errors and the given objective is solved, you move on to the next round.
Watch your code in action: After correcting your bugs, run your script to see your game in action.
